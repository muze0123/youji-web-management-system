# 任务列表 PRD

**文档版本**：v1.0
**页面名称**：任务列表（Task Center）
**触发方式**：全局悬浮入口 / 各模块导出/导入操作后 Toast 提示「点击查看」跳转
**文档状态**：待评审
**创建日期**：2026-07-05

---

## 1. 页面概述

「任务列表」是平台的全局异步任务中心，以**模态弹窗（Modal）**形式呈现，覆盖两类任务的查询与操作：**导入任务 / 下载任务**。

用户可在此查看任务执行状态、下载处理结果文件、排查失败原因。

---

## 2. 弹窗整体布局

弹窗居中显示，宽度固定，最大高度为视口高度的 80%，超出后内容区可滚动。

```
┌──────────────────────────────────────────────────────┐  ← 弹窗头部
│  任务列表                                       [×]  │
├──────────────────────────────────────────────────────┤  ← Tab 栏
│  [导入任务]  [下载任务]                              │
├──────────────────────────────────────────────────────┤  ← 筛选区
│  任务模块: [全部▼]  操作人: [全部▼]                 │
│  操作时间: [开始时间 ~ 结束时间]   [搜索]  [重置]   │
├──────────────────────────────────────────────────────┤  ← 表格头（吸顶）
│  列头行                                              │
├──────────────────────────────────────────────────────┤  ← 数据行（可滚动）
│  数据行 × N                                          │
├──────────────────────────────────────────────────────┤  ← 分页栏（吸底）
│  分页控件                                            │
└──────────────────────────────────────────────────────┘
```

**滚动区域说明：**
- 弹窗头部、Tab 栏、筛选区：固定不滚动
- 表格数据行区域：独立可滚动（overflow-y: auto）
- 表格表头：吸顶（sticky）
- 分页栏：固定在弹窗底部（不随内容滚动）

**弹窗遮罩：** 弹窗下方有半透明遮罩覆盖全屏，点击遮罩不关闭弹窗，仅点击 `×` 关闭。

---

## 3. 弹窗头部

水平 flex 布局，左侧显示弹窗标题「任务列表」，右侧显示关闭图标 `×`。

- 点击 `×` 关闭弹窗
- 关闭弹窗后，记住本次最后访问的 Tab，下次打开时恢复（会话内有效，刷新页面后重置）

---

## 4. Tab 切换栏

水平排列两个 Tab，底部有整体分割线，激活 Tab 底部蓝色下划线覆盖在分割线上。

| Tab 名称 | 说明 |
|---------|------|
| 导入任务 | 所有通过 Excel 批量导入数据的任务记录 |
| 下载任务 | 所有导出数据生成文件的任务记录 |

**默认激活规则：**

| 进入方式 | 默认激活 Tab |
|---------|------------|
| 从全局入口直接打开 | 导入任务 |
| 从导入操作的 Toast「点击查看」跳转 | 导入任务 |
| 从导出操作的 Toast「点击查看」跳转 | 下载任务 |

**Tab 切换行为：**

- 切换 Tab 时，筛选条件重置为默认值，列表刷新至第 1 页

---

## 5. 筛选区

### 5.1 筛选控件说明

两个 Tab 共用相同的筛选项：

| 控件 | 类型 | 默认值 | 说明 |
|------|------|-------|------|
| 任务模块 | 下拉单选 | 全部 | 选项见 5.2 节；支持【×】清空恢复「全部」|
| 操作人 | 下拉单选 + 模糊搜索 | 全部 | 展示系统中所有成员；支持输入姓名关键词模糊搜索；支持【×】清空 |
| 操作时间 | 日期区间选择器 | 空 | 精确到年月日；格式「开始时间 ~ 结束时间」|

### 5.2 任务模块选项

| 选项值 | 说明 |
|-------|------|
| 全部 | 不筛选，展示所有模块的任务 |
| 订单管理 | 订单相关的导入/导出任务 |
| 异常工单 | 异常工单相关的导入/导出任务 |

> 后续根据系统导出数据来源模块持续迭代补充。

### 5.3 筛选交互规则

| 操作 | 行为 |
|------|------|
| 点击【搜索】 | 应用所有当前筛选条件，列表刷新至第 1 页 |
| 点击【重置】 | 所有筛选项恢复默认值，列表刷新至第 1 页 |
| 各筛选条件之间 | AND 关系，同时生效 |
| 切换 Tab | 筛选条件自动重置为默认值 |

## 6. 导入任务 Tab

### 6.1 表格列定义

| 列名 | 宽度 | 对齐 | 说明 |
|------|------|------|------|
| 模块 | `120px` | 左 | 任务所属业务模块名称（如「订单管理」「异常工单」）|
| 文件名称 | `flex: 1` | 左 | 文件完整名称（如「订单数据_20260101.xlsx」）；超长不截断，支持 hover Tooltip 全显 |
| 操作人 | `90px` | 左 | 操作员工姓名 |

| 任务阶段 | `180px` | 左 | 见 6.2 节 |
| 操作 | `100px` | 左 | 见 6.3 节 |

### 6.2 「任务阶段」列状态说明

| 状态 | 展示内容 | 说明 |
|------|---------|------|

## 8. 默认态与搜索态

### 8.1 默认态

- 进入弹窗时，筛选区所有控件为默认值（任务模块为「全部」，操作人为「全部」，操作时间为空）
- 列表展示当前 Tab 下全部任务记录，按操作时间倒序排列（最新的在最上方）

### 8.2 搜索态

- 任意筛选条件有值时进入搜索态
- 列表按筛选条件过滤展示
- 无匹配结果时展示对应空状态（见第 10 节）

---

## 9. 分页栏

固定于弹窗底部，不参与内容区滚动。

**展示格式：**

```
[ < ] [1] [2] [3] [4] [5] [...] [43] [ > ]   10条/页▼   跳至 [n] 页   共40条记录 第1/8页
```

**交互规则：**

| 操作 | 行为 |
|------|------|
| 点击页码 | 加载对应页数据，保持筛选条件不变 |
| 修改每页条数 | 重新加载，页码重置至第 1 页 |
| 跳页输入超出范围 | Toast 提示「请输入有效页码」|
| 默认每页条数 | 10 条/页 |
| 可选每页条数 | 10 / 20 / 50 |

## 11. 页面跳转规则

## 12. 交互说明

### 12.1 文件名称列 hover 全显

文件名称列内容超出列宽时显示省略号，鼠标 hover 该单元格时弹出 Tooltip 展示完整文件名称。

### 12.2 下载文件行为

| 情况 | 行为 |
|------|------|
| 点击【下载文件】（有效期内）| 浏览器直接下载文件，Toast 提示「开始下载」|
| 点击【下载文件】（已过期，30天后）| 不触发下载，hover 显示「文件已过期，无法下载」Tooltip |
| 点击【下载错误数据】| 浏览器直接下载错误数据文件，Toast 提示「开始下载」|

### 12.3 处理中状态刷新

- 列表中「处理中」状态的任务，每隔 10 秒自动刷新该行数据，更新进度百分比
- 不影响其他行数据和分页状态
- 若刷新时发现任务已完成，该行自动切换为对应完成状态

### 12.4 关闭弹窗规则

- 点击右上角 `×` 关闭弹窗
- 点击弹窗外遮罩区域**不关闭**弹窗
- 关闭后记住本次最后访问的 Tab，下次打开时恢复（会话内有效）

## 13. 状态说明

### 13.1 导入任务状态枚举

| 状态值 | 展示文案 | 说明 |
|-------|---------|------|
| processing | `处理中 N%` | 系统正在处理导入文件 |
| success | `总数N条，成功N条，失败0条` | 全部导入成功 |
| partial_fail | `总数N条，成功N条，失败N条` | 部分行校验失败 |
| fail | `导入失败` | 整个任务执行失败 |

### 13.2 下载任务状态枚举

| 状态值 | 展示文案 | 说明 |
|-------|---------|------|
| processing | `处理中 N%` | 系统正在生成导出文件 |
| success | `导出成功` | 文件生成成功，可下载 |
| fail | `导出失败` | 文件生成失败 |
| expired | 文件已过期（仅 hover 显示）| 导出成功但超过 30 天保留期 |

## 14. 异常场景处理

| 场景 | 表现 | 处理方案 |
|------|------|---------|
| 任务列表数据加载失败 | 列表区域显示「加载失败，点击重试」占位 | 点击触发重新请求列表接口 |
| 下载文件接口失败 | Toast 提示「下载失败，请稍后重试」| — |
| 下载错误数据接口失败 | Toast 提示「下载失败，请稍后重试」| — |
| 处理中状态轮询请求失败 | 静默失败，不提示，下次轮询继续 | — |
| 网络断开时执行操作 | Toast 提示「网络异常，请检查网络后重试」| — |
| 跳页输入非法值 | Toast 提示「请输入有效页码」，不执行跳转 | — |

## 15. 交互规则汇总

| 规则 | 说明 |
|------|------|
| Tab 切换重置筛选 | 切换 Tab 时所有筛选条件自动重置为默认值 |
| 关闭弹窗记住 Tab | 会话内记住最后访问的 Tab，刷新页面后重置 |
| 点击遮罩不关闭 | 仅点击 `×` 可关闭弹窗 |
| 列表按操作时间倒序 | 默认最新任务排在最上方 |
| 处理中自动刷新 | 每 10 秒轮询一次，仅更新处理中的行，不影响其他状态 |
| 下载文件可重复下载 | 有效期内【下载文件】按钮可多次点击，每次均触发下载 |
| 文件过期 30 天 | 导出成功后超过 30 天，下载链接失效变灰，hover 提示已过期 |
| 部分失败才显示下载错误 | 仅「部分失败」状态的导入任务操作列才出现【下载错误数据】|
| 全部成功或处理失败无操作 | 导入任务全部成功或处理失败时，操作列均显示 `——` |

## 16. 数据说明

| 数据项 | 规则 |
|-------|------|
| 文件保留期 | 下载任务导出文件最多保留 30 天，到期自动清除 |
| 进度百分比 | 由后端实时计算返回，前端直接展示，单位 `%`，整数显示 |
| 操作时间格式 | `YYYY-MM-DD HH:mm:ss`，精确到秒 |
| 错误数据文件命名 | `{模块名}导入错误数据_{YYYYMMDDHHMMSS}.xlsx` |
| 导出文件命名 | `{模块名}导出_{YYYYMMDDHHMMSS}.xlsx` |
| 列表默认排序 | 按操作时间倒序，最新记录在最上方 |
| 操作人 | 展示执行该任务时的登录账号姓名 |
| 任务模块 | 由触发导入/导出操作的业务模块决定，由后端写入，前端直接展示 |
