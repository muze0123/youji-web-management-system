**文档状态**：待评审
**创建日期**：2026-07-09

---

## 1. 页面概述

优惠券管理是后台运营配置模块的子功能，供运营人员进行优惠券全生命周期管理，包括：创建券模板、配置发放方式（手动/注册赠券/活动领券）、查询发放记录及核销统计数据。

**功能构成：**

| 功能模块 | 说明 |
|---------|------|
| 券模板列表 | 查看、搜索、启停、删除已创建的券模板 |
| 创建/编辑券 | 配置券种、面额、门槛、有效期、适用范围、发放量 |
| 发放券 | 手动发券（指定/批量）、注册赠券配置、活动领券配置 |
| 核销统计 | 发放量、使用量、核销率、抵扣总额等运营指标 |

## 3. Tab 一：券模板管理

### 3.1 默认态布局

```
┌─────────────────────────────────────────────────────────────┐
│  [搜索券名称]  券种:[全部▼]  状态:[全部▼]  [查询] [重置]   │  ← 筛选区
├─────────────────────────────────────────────────────────────┤
│  [创建优惠券]                                               │  ← 工具栏
├─────────────────────────────────────────────────────────────┤
│  券名称  券种  面额  门槛  适用范围  有效期  状态  操作      │  ← 表头
│  数据行 × N                                                 │  ← 列表
├─────────────────────────────────────────────────────────────┤
│  分页控件                                                   │
└─────────────────────────────────────────────────────────────┘
```

### 3.2 筛选区

| 控件 | 类型 | 说明 |
|------|------|------|
| 搜索券名称 | 输入框 | 关键词模糊搜索券名称 |
| 券种 | 下拉单选 | 全部 / 立减券 / 折扣券 / 免费时长券 |
| 状态 | 下拉单选 | 全部 / 启用 / 停用 |
| 【查询】| 主按钮 | 应用筛选条件，列表刷新至第 1 页 |
| 【重置】| 次按钮 | 清空所有筛选项，刷新列表 |

### 3.3 列表字段说明

| 列名 | 说明 |
|------|------|
| 券名称 | 券的展示名称 |
| 券种 | 立减券 / 折扣券 / 免费时长券 |
| 面额/折扣/时长 | 立减券显示「¥X」；折扣券显示「X折（最高抵扣¥Y）」；免费时长券显示「免费Xh」 |
| 使用门槛 | 「满¥X可用」或「无门槛」 |
| 适用范围 | 仅寄存 / 仅寄送 / 通用 |
| 有效期规则 | 「领取后N天」或「YYYY-MM-DD 至 YYYY-MM-DD」 |
| 发放量 | 总库存 / 已发放 / 剩余（格式：「已发N / 总量N」）|
| 状态 | 启用（绿）/ 停用（灰）Toggle 开关 |
| 操作 | 编辑 / 发放 / 删除 |

### 3.4 状态 Toggle 交互

| 操作 | 行为 |
|------|------|
| 点击 Toggle 关闭（启用→停用）| 弹出确认弹窗：「停用后该券将无法被领取和使用，确认停用？」；确认后 Toast「停用成功」 |
| 点击 Toggle 开启（停用→启用）| 直接切换为启用，Toast「已启用」 |

### 3.5 操作列

| 操作 | 行为 |
|------|------|
| 编辑 | 弹出「编辑优惠券」侧边抽屉，预填当前配置；**已有用户持有的券，编辑时面额/折扣/门槛不可修改，仅可改名称/状态/发放量** |
| 发放 | 跳转「发放券」弹窗（见 Tab 二）|
| 删除 | 弹出确认弹窗：「删除后不可恢复，已发放给用户的券不受影响，确认删除？」；确认后删除，Toast「删除成功」 |

### 3.6 列表分页

- 默认每页 10 条；可切换 10 / 20 / 50 条/页
- 格式：页码导航 + 每页条数 + 跳页 + 共N条 第X/Y页

## 4. 创建 / 编辑优惠券抽屉

**触发方式：**
- 点击工具栏【创建优惠券】→ 右侧滑出抽屉，标题「创建优惠券」
- 点击列表行「编辑」→ 右侧滑出抽屉，标题「编辑优惠券」，预填数据

### 4.1 表单字段说明

| 字段 | 必填 | 控件 | 校验规则 |
|------|------|------|---------|
| 券名称 | 是 | 输入框，最多 20 字 | 不可为空；不可与已有券名重复 |
| 券种 | 是 | 单选：立减券 / 折扣券 / 免费时长券 | 创建后**不可修改** |
| 面额（立减券）| 是 | 数字输入框，单位「元」| 正整数或保留2位小数，最小 0.01 |
| 折扣率（折扣券）| 是 | 数字输入框，如「8折」即填「80」| 1-99 整数 |
| 最高抵扣金额（折扣券）| 否 | 数字输入框，单位「元」，空则无上限 | 正数 |
| 免费时长（免费时长券）| 是 | 数字+单位（小时/分钟）| 正整数 |
| 使用门槛 | 是 | 单选：无门槛 / 满X元可用；选满X时输入X | X 为正整数或2位小数 |
| 适用范围 | 是 | 多选：仅寄存 / 仅寄送 / 通用 | 至少选一项 |
| 有效期类型 | 是 | 单选：领取后N天有效 / 固定起止日期 | — |
| 有效期-领取后N天 | 条件必填 | 数字输入框，正整数 | 选「领取后N天」时必填 |
| 有效期-固定起止 | 条件必填 | 日期区间选择器，精确到日 | 起始日期不可早于当天 |
| 发放总量 | 是 | 数字输入框（库存）| 正整数，最小 1 |
| 状态 | 是 | 单选：启用 / 停用，默认启用 | — |
| 备注 | 否 | 多行文本框，最多 200 字 | — |

### 4.2 券种切换联动

选择不同券种时，「面额/折扣/时长」字段动态切换：

| 券种 | 展示字段 | 隐藏字段 |
|------|---------|---------|
| 立减券 | 面额（¥）| 折扣率、最高抵扣、免费时长 |
| 折扣券 | 折扣率（%）+ 最高抵扣（¥，可选）| 面额、免费时长 |
| 免费时长券 | 免费时长（h/min）| 面额、折扣率、最高抵扣 |

### 4.3 提交逻辑

| 操作 | 行为 |
|------|------|
| 点击【保存】| 全量校验，不通过则字段标红并提示；通过则保存，Toast「创建成功」/「修改成功」，抽屉关闭，列表刷新 |
| 点击【取消】或关闭抽屉 | 弹出「确认放弃编辑？」确认弹窗；确认后关闭，已填数据丢失 |

## 5. Tab 二：发放记录（含发放操作入口）

### 5.1 发放记录列表默认态

```
┌──────────────────────────────────────────────────────────────┐
│  [搜索用户名/手机]  券种:[全部▼]  时间:[开始~结束]  [查询][重置]│
├──────────────────────────────────────────────────────────────┤
│  [手动发放]  [配置注册赠券]  [配置活动领券]                  │  ← 工具栏
├──────────────────────────────────────────────────────────────┤
│  用户  券名称  券种  面额  发放方式  发放时间  状态  操作      │  ← 表头
│  数据行 × N                                                  │
├──────────────────────────────────────────────────────────────┤
│  分页控件                                                    │
└──────────────────────────────────────────────────────────────┘
```

### 5.2 发放记录列字段

| 列名 | 说明 |
|------|------|
| 用户 | 用户名 + 手机号（脱敏，如 `138****0002`）|
| 券名称 | 券的展示名称 |
| 券种 | 立减券 / 折扣券 / 免费时长券 |
| 面额/折扣/时长 | 同券模板列表格式 |
| 发放方式 | 手动发放 / 注册赠券 / 活动领取 |
| 发放时间 | `YYYY-MM-DD HH:mm:ss` |
| 券状态 | 未使用 / 已使用 / 已过期 / 已退回 |
| 操作人 | 执行手动发放的管理员姓名（自动发放显示「系统」）|

### 5.3 手动发放弹窗

**触发**：点击工具栏【手动发放】

**弹窗字段：**

| 字段 | 说明 |
|------|------|
| 选择券模板 | 下拉搜索，仅展示状态为「启用」的券 |
| 发放方式 | 单选：指定用户 / 批量用户 |
| 指定用户 | 输入用户名或手机号模糊搜索，多选，展示已选列表 |
| 批量用户 | 上传 Excel 文件（模板下载）；字段：手机号；上传后预览解析结果 |
| 每人发放数量 | 数字，正整数，默认 1 |
| 发放备注 | 选填，最多 100 字（留痕，如「客服补偿-订单XX」）|

**提交逻辑：**

| 操作 | 行为 |
|------|------|
| 点击【确认发放】| 弹出二次确认弹窗：「确认向 N 位用户各发放 M 张【券名称】？」；确认后执行，Toast「发放成功」 |
| 发放成功 | 发放记录列表新增对应条目；每次发放留痕（操作人、时间、备注）|
| 批量文件解析失败 | 提示「文件格式有误，请下载模板后重新填写」|

### 5.4 配置注册赠券弹窗

**触发**：点击工具栏【配置注册赠券】

**功能**：配置新用户注册时自动赠送的券。

| 字段 | 说明 |
|------|------|
| 启用/停用 | Toggle 开关，关闭则新注册用户不赠券 |
| 赠送的券 | 多选（可配置多张），每行展示「券名称 × N张」|
| 添加券 | 点击「+ 添加」选择已启用的券模板，输入数量 |

**提交**：点击【保存配置】，Toast「保存成功」。

### 5.5 配置活动领券弹窗

**触发**：点击工具栏【配置活动领券】

| 字段 | 说明 |
|------|------|
| 活动名称 | 最多 20 字 |
| 关联券模板 | 单选已启用的券，填写每人可领数量 |
| 活动有效期 | 日期区间，用户只能在此期间领取 |
| 每人限领次数 | 正整数，默认 1 |
| 总领取上限 | 可选，正整数；留空则不限 |
| 活动状态 | 启用 / 停用 |

## 6. Tab 三：核销统计

### 6.1 页面结构

```
┌──────────────────────────────────────────────────────────────┐
│  时间范围:[全部▼]  券种:[全部▼]  券名称:[全部▼]  [查询][重置]│  ← 筛选区
├──────────────────────────────────────────────────────────────┤
│  [汇总指标卡片行]                                            │
│  发放量 | 领取量 | 使用量 | 核销率 | 抵扣总额              │
├──────────────────────────────────────────────────────────────┤
│  [使用量/抵扣趋势折线图（Chart.js）]                         │
├──────────────────────────────────────────────────────────────┤
│  券明细数据列表                                              │
│  券名称  券种  发放量  领取量  使用量  核销率  抵扣总额      │
└──────────────────────────────────────────────────────────────┘
```

### 6.2 汇总指标卡片

| 指标 | 说明 |
|------|------|
| 发放量 | 当前筛选条件下已发放的券总数（张）|
| 领取量 | 活动领券方式中用户主动领取数（张）|
| 使用量 | 已核销（下单抵扣）的券数（张）|
| 核销率 | 使用量 / 发放量 × 100%（保留1位小数）|
| 抵扣总额 | 券抵扣的金额总和（¥，保留2位小数）|

### 6.3 趋势图

- 图表类型：折线图（Chart.js）
- X 轴：日期（按筛选时间粒度：按天/按周/按月）
- Y 轴左：使用量（张）；Y 轴右：抵扣总额（¥）
- 图例：使用量 / 抵扣总额
- Hover 数据点：Tooltip 展示日期 + 两项指标数值

### 6.4 券明细表格

| 列名 | 说明 |
|------|------|
| 券名称 | — |
| 券种 | — |
| 发放量 | 该券已发放张数 |
| 领取量 | 该券主动领取张数 |
| 使用量 | 该券已核销张数 |
| 核销率 | 使用量/发放量（%）|
| 抵扣总额 | 该券抵扣金额之和（¥）|

- 支持按「使用量」「核销率」「抵扣总额」列头点击排序
- 支持导出 Excel（当前筛选条件下的明细数据）

## 8. 状态说明

### 8.1 券模板状态

| 状态 | 说明 | 影响 |
|------|------|------|
| 启用 | 正常可发放、可被用户使用 | — |
| 停用 | 不可新发放，已发放的券不可使用 | 已持有用户的券同步失效 |

### 8.2 发放记录中的券状态

| 状态 | 说明 |
|------|------|
| 未使用 | 用户持有，未使用 |
| 已使用 | 已在订单中核销 |
| 已过期 | 超有效期未用 |
| 已退回 | 用券订单退款后券退回（回到未使用状态）|

## 9. 异常场景处理

| 场景 | 表现 | 处理方案 |
|------|------|---------|
| 列表/统计数据加载失败 | 内容区显示「加载失败，点击重试」| 点击重新请求 |
| 创建券名称重复 | 券名称字段标红，提示「券名称已存在」| — |
| 批量发放 Excel 格式错误 | 上传后提示「文件格式有误，请下载模板」| — |
| 批量发放含无效手机号 | 解析预览时标红无效行，显示「第N行手机号无效」| 支持修正后重新上传 |
| 发放量超过剩余库存 | 提交时校验，提示「剩余库存不足，当前剩余N张」| — |
| 删除已有用户持有的券模板 | 确认弹窗说明「已发放给用户的券不受影响」| 删除后模板不可再发放 |
| 网络断开时提交 | Toast「网络异常，请检查网络后重试」| — |

## 10. 交互规则汇总

| 规则 | 说明 |
|------|------|
| 券种一旦创建不可修改 | 创建成功后，编辑时券种字段置灰不可编辑 |
| 已有持有用户的券核心字段不可编辑 | 面额/折扣/门槛字段在已发放后锁定 |
| 停用券影响已持有用户 | 停用后用户端该券变为不可用，下单时无法选用 |
| 发放需二次确认 | 所有手动发放均弹出确认弹窗，展示发放范围和数量 |
| 发放记录完整留痕 | 每次发放记录：操作人、发放方式、时间、备注、受益用户列表 |
| 核销统计与财务口径一致 | 抵扣总额纳入财务收入抵减口径 |

## 11. 数据说明

| 数据项 | 规则 |
|-------|------|
| 券模板 ID | 系统自动生成，唯一 |
| 发放量统计 | 实时统计，每次发放后更新 |
| 核销率 | 使用量 / 发放量 × 100%，保留1位小数 |
| 抵扣总额 | 仅统计「已使用」状态的券产生的抵扣金额；已退回的券对应金额扣回 |
| 手机号脱敏 | 列表展示时中间4位替换为`****` |
| 有效期-领取后N天 | 从用户领取时间起算，精确到秒，过期后自动变为「已过期」|
