**所属模块**：系统设置 / 消息设置
**文档状态**：待评审

---

## 1. 页面概述

消息设置页面用于配置系统内各类消息通知的推送方式与推送对象。管理员可为每种消息类型独立开启或关闭气泡提醒（桌面端弹窗）和消息中心提醒，并指定通知的推送角色。同时支持绑定微信公众号，开启公众号推送渠道。

**访问权限**：需拥有「消息设置 - 查看」功能权限，无权限时菜单隐藏，不可访问。

## 3. 提示横幅

### 3.1 内容与样式

页面顶部展示警告风格的提示横幅（warning 底色 + 左侧 warning 色竖条），说明消息设置的基本规则。

**横幅文案：**

- 1、设置启用时，会受到系统的弹窗提醒或消息中心提醒。
- 2、开启气泡提醒时默认开启消息中心提醒，关闭消息中心提醒默认关闭气泡提醒。

### 3.2 横幅右侧按钮

| 按钮 | 图标 | 触发行为 |
|------|------|---------|
| 绑定公众号 | `message-circle` | 弹出「微信提醒设置」弹窗，见第 8 节 |
| 消息中心 | `bell` | 跳转至「消息中心」页面 |

## 4. 消息配置表格

### 4.1 表格列说明

| 列名 | 宽度 | 说明 |
|------|------|------|
| 消息标题 | 140px | 消息类型的名称，加粗显示 |
| 消息说明 | 自适应 | 该消息类型的触发场景说明 |
| 气泡提醒 | 130px | Toggle 开关，控制桌面端右上角气泡弹窗通知（5 秒自动消失）；表头含 `?` 提示图标 |
| 消息中心提醒 | 130px | Toggle 开关，控制该消息是否记录到消息中心 |
| 通知方式 | 130px | 只读展示，显示该消息的推送渠道（系统消息 / 公众号推送） |
| 通知对象 | 130px | 只读展示，显示该消息的推送角色 |
| 操作 | 80px | 「编辑」链接，点击弹出编辑弹窗 |

### 4.2 气泡提醒开关

| 操作 | 行为 |
|------|------|
| 点击开关（开启→关闭）| 直接关闭，Toast 提示「操作成功」 |
| 点击开关（关闭→开启）| 开启气泡提醒，同时自动开启消息中心提醒（联动），Toast 提示「操作成功」 |

### 4.3 消息中心提醒开关

| 操作 | 行为 |
|------|------|
| 点击开关（开启→关闭）| 关闭消息中心提醒，同时自动关闭气泡提醒（联动），Toast 提示「操作成功」 |
| 点击开关（关闭→开启）| 直接开启，Toast 提示「操作成功」 |

### 4.4 联动规则

> **核心规则**：气泡提醒依附于消息中心提醒。
>
> - 开启气泡提醒 → 自动开启消息中心提醒（气泡依赖消息中心）
> - 关闭消息中心提醒 → 自动关闭气泡提醒（皮之不存，毛将焉附）

| 操作 | 气泡提醒 | 消息中心提醒 |
|------|---------|------------|
| 开启气泡 | → 开 | → 自动开 |
| 关闭气泡 | → 关 | 不变 |
| 开启消息中心 | 不变 | → 开 |
| 关闭消息中心 | → 自动关 | → 关 |

### 5.1 通知方式说明

| 通知方式 | 说明 | 是否必选 |
|---------|------|---------|
| 系统消息 | 通过系统内部消息中心推送，默认必选 | 是（始终开启，不可取消） |
| 公众号推送 | 通过微信公众号模板消息推送，需先绑定公众号 | 否（可选，勾选前需绑定公众号） |

### 5.2 通知对象（角色）

| 角色 | 说明 |
|------|------|
| 所属运营 | 订单或异常件当前关联的运营人员 |
| 运营主管 | 具有运营主管角色的成员 |
| 系统管理员 | 具有系统管理员角色的成员 |

## 6. 编辑消息配置弹窗

### 6.1 弹窗标题

格式：「编辑 - 消息标题」（如「编辑 - 订单分配」）

### 6.2 弹窗字段

| 字段 | 必填 | 说明 |
|------|------|------|
| 推送角色 | 是 | 复选框组，选项：「所属运营」「运营主管」「系统管理员」；至少选一项 |
| 推送方式 | — | 复选框组，选项：「系统消息」「公众号推送」；系统消息始终选中且不可取消（disabled）；公众号推送可选 |

### 6.3 公众号推送勾选校验

当用户勾选「公众号推送」时：
- 若尚未绑定公众号：弹出 Toast 提示「请先绑定公众号」，自动打开「微信提醒设置」弹窗，公众号推送复选框不勾选
- 若已绑定公众号：正常勾选

### 6.4 校验规则

| 校验项 | 规则 | 错误提示 |
|--------|------|---------|
| 推送角色 | 不可为空（至少选一项） | Toast 提示「配置角色不能为空」 |

### 6.5 提交行为

- 点击【确定】：若校验不通过，Toast 错误提示，弹窗不关闭；校验通过后，关闭弹窗，表格数据刷新，Toast 提示「保存成功」
- 点击【取消】或右上角 `×`：关闭弹窗，不保存

## 7. 页面操作汇总

| 操作 | 触发方式 | 行为 |
|------|---------|------|
| 切换气泡提醒 | 点击 Toggle | 直接切换；开启时联动开启消息中心提醒 |
| 切换消息中心提醒 | 点击 Toggle | 直接切换；关闭时联动关闭气泡提醒 |
| 编辑消息配置 | 点击行末「编辑」 | 弹出编辑弹窗，修改推送角色和推送方式 |
| 绑定公众号 | 点击横幅右侧「绑定公众号」 | 弹出微信提醒设置弹窗（见第 8 节） |
| 跳转消息中心 | 点击横幅右侧「消息中心」 | 跳转至消息中心页面 |

## 8. 微信提醒设置弹窗

### 8.1 弹窗内容

- 标题：「微信提醒设置」
- 内容区居中展示：
  - 公众号二维码占位图（120×120px）
  - 引导文案：「请用微信"扫一扫"」
  - 说明文案：「微信扫码关注"游寄"微信公众号」
  - 功能说明卡片（page 底色背景）：
    - 「绑定微信公众号，您将获得以下功能：」
    - ✅ 柜机异常、行李超时等消息通知
    - ✅ 订单状态变更、退款审核等消息通知

### 8.2 弹窗尺寸

宽度 460px，最大高度 70vh，居中展示，遮罩背景 `rgba(0,0,0,.45)`。

### 8.3 关闭方式

- 点击遮罩区域
- 点击右上角 `×` 关闭按钮

## 9. 默认态

- 进入页面时，所有 14 条消息类型默认同时开启气泡提醒和消息中心提醒
- 所有消息的通知方式和通知对象按第 5 节清单预置
- 若未绑定公众号，部分消息类型的「公众号推送」仅为可选项，需绑定后方可生效

## 11. 异常场景处理

| 场景 | 表现 | 处理方案 |
|------|------|---------|
| 编辑保存时校验失败 | Toast 提示「配置角色不能为空」 | 弹窗不关闭，用户补选后重新提交 |
| 未绑定公众号时勾选公众号推送 | Toast 提示「请先绑定公众号」 | 自动弹出微信提醒设置弹窗，复选框不勾选 |
| 页面数据加载失败 | 表格区域显示空状态 | 提示用户刷新页面重试 |

## 12. 交互规则汇总

| 规则 | 说明 |
|------|------|
| 气泡→消息中心联动 | 开启气泡自动开启消息中心 |
| 消息中心→气泡联动 | 关闭消息中心自动关闭气泡 |
| 系统消息不可取消 | 编辑弹窗中「系统消息」始终选中且 disabled |
| 公众号推送需先绑定 | 勾选公众号推送时校验绑定状态，未绑定时引导绑定 |
| Toggle 切换即时生效 | 无需二次确认，切换后 Toast 反馈 |
| 推送角色至少一项 | 编辑弹窗保存时校验，为空时阻止提交 |
| 编辑弹窗单独操作 | 每次只能编辑一条消息类型，弹窗标题显示当前编辑的消息标题 |
