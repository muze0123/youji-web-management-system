**创建日期**：2026-07-18

---

## 1. 页面概述

部门管理页面用于管理企业部门组织结构，支持新增、编辑、删除部门，以及设置部门负责人。页面以表格形式展示部门树形结构，支持按部门名称搜索过滤。

**访问权限**：需拥有「部门管理 - 查看」功能权限，无权限时菜单隐藏，不可访问。

---

├──────────────────────────────────────────────────────────────────┤
│  ▼ 杭州渡河科技有限公司（10人）              张三      编辑 删除    │  ← 根节点
│    ▼ 技术部（5人）                          李四      编辑 删除    │  ← 一级部门
│      ▶ 前端组（2人）                        王五      编辑 删除    │  ← 二级部门

## 4. 部门列表表格

### 4.1 表格列说明

| 列名 | 宽度 | 说明 |
|------|------|------|
| 部门名称 | min-width: 260px | 树形展示，含展开/折叠箭头；根节点加粗显示；显示部门人数 |
| 部门人数 | 120px | 该部门直属人数 + 所有子部门人数之和 |
| 负责人 | 160px | 部门负责人姓名（多选，可多个） |
| 操作 | 200px | 编辑、删除链接 |

### 4.2 树形结构展示

| 元素 | 说明 |
|------|------|
| 展开/折叠图标 | `▶` 表示收起，`▼` 表示展开，点击切换折叠状态；叶子节点隐藏图标 |
| 部门名称 | 根节点加粗（text-ink-title），普通节点正常（text-ink-body） |
| 人数标注 | 显示「（N人）」，N = 本节点直属人数 + 所有子部门人数之和 |
| 缩进 | 子部门缩进显示，形成视觉层级 |

### 4.3 操作列

| 操作 | 说明 |
|------|------|
| 编辑 | 蓝色链接，点击弹出编辑部门弹窗 |
| 删除 | 蓝色链接，点击弹出删除确认对话框 |

## 5. 添加/编辑部门弹窗

### 5.1 弹窗布局

```
┌──────────────────────────────────────────────────────────────────┐
│  [添加部门 / 编辑部门]                    [×]                     │  ← 标题栏
├──────────────────────────────────────────────────────────────────┤
│                                                                  │
│  *部门名称： [________________________]                          │
│                                                                  │
│  *上级部门： [全部部门▼]                                          │
│                                                                  │
│  负责人：    [请选择负责人 ▼]                                     │
│                                                                  │
├──────────────────────────────────────────────────────────────────┤
│                  [取消]          [确定]                           │  ← 底部按钮
└──────────────────────────────────────────────────────────────────┘
```

### 5.2 表单字段

| 字段 | 必填 | 类型 | 说明 |
|------|------|------|------|
| 部门名称 | 必填 | 文本输入 | 最大长度 20 字符；输入校验：非空、不重复 |
| 上级部门 | 必填 | 下拉选择 | 默认「全部部门」（即根节点下）；可选择任意已有部门作为上级 |
| 负责人 | 选填 | 多选 | 从成员列表中选择；支持多选；显示为 Tag 形式 |

### 5.3 交互

| 操作 | 响应 |
|------|------|
| 点击确定 | 校验表单；校验通过后保存，关闭弹窗，刷新列表 |
| 点击取消 | 关闭弹窗，不保存 |
| 部门名称输入错误 | 显示红色错误提示 |

---

## 7. 业务规则

- 部门支持多级树形结构；
- 删除部门时，该部门及其所有子部门将被级联删除；
- 删除部门时，该部门的成员将被移除（解除部门关联）；
- 根节点（公司名称）不可删除；
- 部门名称不可重复；
- 负责人支持多选；
- 仅管理员可操作本模块。
