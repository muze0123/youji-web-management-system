## 一、页面概述

### 1.1 页面定位
本页面为游寄后台管理系统的布局规范，定义了整个系统的标准布局结构，包含顶部导航栏、左侧侧边栏、主内容区三个核心区域，适用于所有后台管理页面。

### 1.2 用户场景
- 用户登录后台管理系统后，通过顶部导航栏查看系统名称、任务列表、消息中心、用户信息
- 用户通过左侧导航栏切换不同功能模块
- 用户在主内容区查看和操作具体业务功能

### 2.2 区域详细布局

#### 2.2.1 顶部导航栏（Top Navbar）
- **布局位置**：页面最顶部，宽度100%，高度固定
- **排列方式**：左右分布
- **内容结构**：
  - 左侧区域：平台logo + 平台名称
  - 右侧区域：任务列表图标 + 消息中心图标 + 用户头像

#### 2.2.2 左侧侧边栏（Sidebar）
- **布局位置**：顶部导航栏下方左侧，宽度固定，高度占满剩余屏幕高度
- **排列方式**：从上到下，一级菜单和二级菜单
- **内容结构**：
  - 一级菜单：工作台、数据运营、找达人、达人管理
  - 二级菜单：达人库、智能推荐达人、商品找达人、达人榜单

#### 2.2.3 主内容区（Main Content Area）
- **布局位置**：顶部导航栏下方右侧，占据剩余空间
- **排列方式**：内容居中显示
- **内容结构**：
  - 外层背景：浅灰色
  - 内部白色内容容器：圆角设计，max-width: 1280px，padding: 20px

## 五、边界条件汇总

### 5.1 顶部导航栏边界条件
1. 顶部导航栏高度固定，宽度100%
2. 平台名称长度超过20个字符时，显示为"..."省略号
3. 未读消息数量为0时，不显示红色角标
4. 未读消息数量超过99时，显示"99+"
5. 用户头像加载失败时，显示默认用户头像

### 5.2 左侧侧边栏边界条件
1. 左侧侧边栏宽度固定，高度占满顶部导航栏下方的剩余屏幕高度
2. 一级菜单名称长度超过10个字符时，显示为"..."省略号
3. 二级菜单名称长度超过15个字符时，显示为"..."省略号
4. 菜单层级最多支持2级
5. 菜单项数量为0时，显示"暂无菜单"
6. 当前选中的一级菜单使用蓝色图标+蓝色文字
7. 当前选中的二级菜单使用浅蓝色背景+蓝色文字
8. "New"标签仅在isNew为true时显示

### 5.3 主内容区边界条件
1. 主内容区max-width: 1280px
2. 主内容区padding: 20px
3. 内容容器圆角约8px
4. 内容容器背景为白色
5. 外层背景为浅灰色

### 5.4 响应式边界条件
1. 屏幕宽度小于1280px时，主内容区max-width自适应
2. 屏幕宽度小于768px时，左侧侧边栏可收起
3. 屏幕宽度小于480px时，顶部导航栏右侧图标隐藏

## 八、注意事项

### 8.1 整体布局注意事项
1. 页面采用"上-左-右"三栏布局
2. 顶部导航栏宽度100%，高度固定，悬浮在页面最上方
3. 左侧侧边栏宽度固定，高度占满顶部导航栏下方的剩余屏幕高度
4. 主内容区占据页面剩余的全部空间

### 8.2 顶部导航栏注意事项
1. 导航栏背景为纯白色，底部有一条细浅灰色边框
2. 左侧包含平台Logo和平台名称
3. 右侧包含任务列表图标、消息中心图标、用户头像图标
4. 三个图标横向等间距排列，位于导航栏最右侧
5. 图标均为圆形，风格统一，颜色为低饱和度的灰色
6. 消息中心图标右上角有红色圆形角标显示未读消息数量

### 8.3 左侧侧边栏注意事项
1. 侧边栏背景为纯白色，右侧有一条细浅灰色边框
2. 所有菜单项均为左对齐，图标与文字有固定间距
3. 文字均为无衬线字体，未选中状态为深灰色，选中状态为蓝色
4. 一级菜单和二级菜单有明确的视觉层级区分
5. 选中的一级菜单使用蓝色图标+蓝色文字
6. 选中的二级菜单使用浅蓝色背景+蓝色文字
7. 子菜单有缩进，和一级菜单形成视觉层级
8. "New"标签为圆角小标签，颜色醒目，突出新功能

### 8.4 主内容区注意事项
1. 外层背景为浅灰色
2. 内部白色内容容器带有圆角（约8px）
3. 内容容器有padding: 20px的内边距
4. 内容容器最大宽度限制为max-width: 1280px
5. 内容居中显示

### 8.5 色彩规范注意事项
1. 以白、浅灰、深灰为基础色调
2. 蓝色作为功能色（选中状态、新标签、强调元素）
3. 红色作为通知警示色（消息角标）
4. 色彩体系简洁清晰，符合后台系统的严谨性

### 8.6 字体规范注意事项
1. 统一使用无衬线字体（如思源黑体、微软雅黑类）
2. 层级清晰：系统标题>菜单文字>内容说明文字

### 8.7 间距规范注意事项
1. 所有元素（图标与文字、菜单项之间、内容区边距）都有规整的间距
2. 布局留白合理，视觉呼吸感充足

### 8.8 边框与圆角注意事项
1. 页面分隔线使用细浅灰色边框
2. 内容容器、图标使用圆角
3. 整体风格统一，偏向现代简约的企业产品设计
