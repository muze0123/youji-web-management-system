## 一、页面概述

### 1.1 页面定位

本规范定义游寄后台管理系统的统一布局结构，所有业务页面均遵循此规范。系统采用经典后台三段式布局：顶部导航栏 + 左侧侧边栏 + 主内容区。

### 1.2 用户场景

- 用户登录后台管理系统后，通过顶部导航栏查看系统名称、任务列表、消息中心、用户信息；
- 用户通过左侧导航栏切换不同功能模块；
- 用户在主内容区查看和操作具体业务功能。

### 2.2 区域详细布局

#### 2.2.1 顶部导航栏（Top Bar）

- **布局位置**：页面最顶部，宽度 100%，高度固定（h-14 = 56px）
- **排列方式**：左侧 Logo + 系统名称，右侧任务列表 + 消息中心 + 用户头像
- **背景**：白色（bg-white），底部边框（border-b border-line）
- **内容结构**：

| 区域 | 组件 | 说明 |
|------|------|------|
| 左侧 | Logo | 方形图标（w-8 h-8），内有"游"字，bg-primary，白色文字 |
| 左侧 | 系统名称 | "游寄后台管理系统"，text-[15px]，font-semibold，text-ink-title |
| 右侧 | 任务列表按钮 | Lucide clipboard-list 图标，点击弹出任务列表抽屉 |
| 右侧 | 消息中心按钮 | Lucide bell 图标，右上角红色角标显示未读数量 |
| 右侧 | 用户头像按钮 | 用户头像（圆形），hover 弹出用户气泡卡片 |

#### 2.2.2 左侧侧边栏（Sidebar）

- **布局位置**：顶部导航栏下方左侧，宽度固定（w-[220px]），高度占满剩余屏幕高度
- **背景**：白色（bg-white），右侧边框（border-r border-line）
- **排列方式**：从上到下，一级菜单（可展开二级菜单）
- **菜单结构**：

| 一级菜单 | 二级菜单 | 图标 |
|----------|----------|------|
| 工作台 | — | layout-dashboard |
| 数据概览 | — | bar-chart-3 |
| 订单管理 | 订单列表、售后列表 | package |
| 柜机监控 | — | monitor |
| 对账中心 | — | file-text |
| 运营配置 | 优惠券管理、价格配置、目的地配置 | sliders-horizontal |
| 用户管理 | 用户列表、投诉建议 | users |
| 系统设置 | 成员管理、部门管理、角色管理、消息设置、账号信息、操作日志 | settings |

**菜单样式**：

| 状态 | 样式 |
|------|------|
| 一级菜单默认 | 深灰色文字，灰色图标边框 |
| 一级菜单激活 | 蓝色文字（text-primary），蓝色图标背景（bg-primary），白色图标 |
| 二级菜单默认 | 深灰色文字，缩进 64px |
| 二级菜单激活 | 蓝色文字（text-primary），浅蓝色背景（bg-primary-bg），缩进 64px |

#### 2.2.3 主内容区（Main Content Area）

- **布局位置**：顶部导航栏下方右侧，占据剩余空间
- **背景**：浅灰色（bg-bg-page = #F7F8FA）
- **布局规则**（详见 design.md §6.1）：

| 场景 | 布局方式 |
|------|----------|
| 单区块 | p-4 容器加 min-h-full flex flex-col，卡片加 flex-1 撑满 |
| 多区块（≥2） | p-4 容器不设高度，卡片高度由内容决定，mb-4 间距 |

**内容卡片**：白色背景（bg-white）、圆角（rounded-lg）、边框（border border-line）、内边距（p-5 md:p-6）。

## 三、区域交互说明

### 3.1 顶部导航栏交互

#### 3.1.1 左侧区域

**Logo + 系统名称**

- **组件类型**：Logo + 系统标题
- **外观**：方形蓝色图标（w-8 h-8），内有白色"游"字；右侧蓝色文字"游寄后台管理系统"
- **交互说明**：
  - 点击：跳转到工作台（index.html）
  - 移动端（<768px）：系统名称隐藏，仅显示 Logo

#### 3.1.2 右侧区域

**任务列表按钮**

- **组件类型**：功能图标按钮
- **外观**：圆形浅灰色背景（w-9 h-9），内部 Lucide clipboard-list 图标
- **交互说明**：
  - 点击：从右侧弹出任务列表抽屉（宽度 360px）
  - 角标：显示待处理任务数量，数量为 0 时隐藏
  - hover：背景变浅蓝，图标变蓝

**消息中心按钮**

- **组件类型**：功能图标按钮
- **外观**：圆形浅灰色背景（w-9 h-9），内部 Lucide bell 图标，右上角红色角标显示未读数量
- **交互说明**：
  - 点击：跳转到消息中心页面（消息中心.html）
  - 角标：红色圆形，显示未读消息数量
  - hover：背景变浅蓝，图标变蓝

**用户头像按钮**

- **组件类型**：用户入口图标
- **外观**：圆形头像（实际为首字母图标），尺寸 w-9 h-9
- **交互说明**：
  - hover：弹出用户气泡卡片（包含用户信息、操作入口）
  - 点击：无额外操作（气泡卡片已展示）

### 3.2 左侧侧边栏交互

#### 3.2.1 一级菜单

**交互说明**：

| 操作 | 响应 |
|------|------|
| 点击无二级菜单的一级菜单 | 跳转到对应页面 |
| 点击有二级菜单的一级菜单 | 展开/收起二级菜单，箭头旋转 180° |
| hover | 背景变浅灰（bg-bg-hover） |
| 激活状态 | 蓝色图标背景 + 蓝色文字 |

#### 3.2.2 二级菜单

**交互说明**：

| 操作 | 响应 |
|------|------|
| 点击 | 跳转到对应页面，高亮当前项 |
| hover | 背景变浅灰 |
| 激活状态 | 浅蓝色背景 + 蓝色文字 |

#### 3.2.3 移动端菜单切换

- **组件类型**：移动端菜单按钮
- **外观**：Lucide menu 图标
- **交互说明**：
  - 点击：显示/隐藏侧边栏（移动端 <768px）
  - 点击遮罩层：关闭侧边栏

### 3.3 主内容区交互

#### 3.3.1 内容卡片

- **组件类型**：内容容器
- **外观**：白色背景，圆角（rounded-lg），边框（border border-line），内边距（p-5 md:p-6）
- **交互说明**：纯展示，无交互

## 四、全局组件

### 4.1 任务列表抽屉

- **位置**：右侧滑出，宽度 360px
- **触发**：点击顶部导航栏任务列表按钮
- **内容**：待办任务列表、下载任务列表、任务筛选
- **交互**：点击遮罩关闭，点击任务查看详情

### 4.2 用户气泡卡片

- **位置**：用户头像上方弹出
- **触发**：hover 用户头像按钮
- **内容**：用户姓名、角色、部门；操作入口（任务列表、消息中心、操作日志、退出登录）
- **交互**：hover 显示，离开后延迟关闭（300ms）
